Modularizando seu código com Requirejs
=========

Projeto apresentado no evento Javaneiros 2014.

Pré-requisitos:
- Node.js
- Grunt

Execute:
- node run.js
- Abra o navegador na url: http://localhost:3000

Versão minificada:
- node rundist.js

Minificando:
Navegue para dentro da pasta do projeto e execute o seguinte comando.
- grunt
